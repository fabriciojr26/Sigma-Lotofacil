# Arquitetura

## Front-end

HTML/CSS/JS puro, com animação elétrica em DOM/CSS e integração com APIs serverless.

## Back-end

Cloudflare Pages Functions. O motor roda no back-end:

- parser;
- validação;
- frequência;
- atraso;
- matriz de pares;
- transição N→N+1;
- geração ensemble;
- conferência;
- backtest.

## Dataset interno

O arquivo `functions/_data/lotofacil3719.js` contém os concursos convertidos do XLSX enviado pelo usuário até o concurso 3719.
