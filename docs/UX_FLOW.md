# UX Flow

1. Usuário abre a home do SIGMA NEXUS CORE.
2. O sistema exibe dataset interno carregado.
3. Usuário informa o último sorteio ou envia dataset próprio.
4. Ao ativar o core, aparece a animação elétrica/magnética.
5. O front-end envia a requisição para `/api/generate`.
6. O back-end gera 20 cenários técnicos.
7. A interface exibe cards, métricas, relatório, CSV/PDF.
8. Usuário pode conferir resultado real no módulo de conferência.
